Caso 01: Sua secretária mentiu?

Agora que você sabe as tabelas que existem na base de dados "Pasta", você pode usar cada Ficha (tabela).
Há duas formas de fazer este processo:
Exemplo 01: DESCRIBE nomeDaTabela;
Fará a descrição das informações (campos).
ou
Exemplo 02: SELECT * FROM nomeDaTabela;
O * informa que você quer ver todos os dados (registros) de uma ficha (tabela). Assim você pode ver/ler todos os dados de uma tabela.
Observação 1: Você pode executar o SHOW TABLES; para ver novamente as informações anteriores.
Observação 2: Ao clicar em "Avançar" você irá para ser redirecionado ao relatório final deste caso.
